# Low-level programming lab3
### Вариант 4: Thrift
#### Сборка
+ Linux / Windows:
```
git clone https://github.com/Gramdel/llp_lab3
cd llp_lab3
cmake .
cmake --build .
```
#### Запуск клиента:
+ Linux:
```
./client_exec <hostname> <port>
```
+ Windows:
```
.\client_exec.exe <hostname> <port>
```
#### Запуск сервера:
+ Linux:
```
./server_exec <filename> <port>
```
+ Windows:
```
.\server_exec.exe <filename> <port>
```
